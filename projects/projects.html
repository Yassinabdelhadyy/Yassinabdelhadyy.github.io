    • The Guardian API

    The aim of theguardian package is to effortlessly retrieve information from The Guardian newspaper API. This package is based on two primary concepts:
    -Retrieving the fundamental API data provided by The Guardian.
    -Retrieving the fundamental API data provided by The Guardian along with extracting the article's body, author name, and tags through web scraping.
